Various HTML Tags & Their Handy Uses

  • Go to 'Home'
Heading
Allows the user to create a heading above any block of writing or chart or anything else and helps seperate sections.
Blockquote
Allows the user to insert a large stand alone quote and include citation information alongside it.
Paragraph
Allows the user to create a block of text.
Image
Allows the user to insert any image of their choice.
Abbreviation
Allows the user to create an abbreviated version of a word while still having the full length word show up when it is hovered over.
